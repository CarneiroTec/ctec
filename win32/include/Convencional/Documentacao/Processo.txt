Documentação do módulo Processo
==============================

Arquivos no diretório Fonte/Sistema/Processo
- Processo.ctec — define estruturas e funções relativas ao controle de processos na biblioteca.
- GerenciadorProcesso.ctec — implementa criação/gerenciamento de processos, se aplicável ao sistema.

Resumo
- O módulo `Processo` provê abstrações para iniciar e gerenciar processos externos, agendar tarefas e finalizar execuções de forma consistente com `Sistema.execucao`.

Como usar
- Para iniciar/gerenciar processos, consulte `Sistema.processo` e use as funções oferecidas pelo `GerenciadorProcesso`.

Localização no código
- Fonte/Sistema/Processo/Processo.ctec
- Fonte/Sistema/Processo/GerenciadorProcesso.ctec

Notas
- Implementações específicas de criação/terminação podem depender do sistema operacional; verifique os wrappers no gerenciador.

Fim.

Exemplos práticos (Correto / Errado)

Exemplo correto — iniciar processo e verificar retorno (pseudo-CTEC):


Inteiro pid = Sistema.processo->crie("meu_programa.exe", "arg1 arg2");
Se (pid > 0) {
	Sistema.terminal->exiba_formatado("Processo iniciado: %d\n", pid);
} Senão {
	Inteiro e = Sistema.erro->obtenha_erro();
	Sistema.terminal->exiba_formatado("Erro ao criar processo: %d\n", e);
}


Exemplo errado — não verificar se o processo foi criado:


Sistema.processo->crie("meu_programa.exe", "");
// Não checa se houve falha


Por que é errado
- A criação de processos pode falhar por diversos motivos (permissões, inexistência do executável, recursos insuficientes). Sempre verifique o retorno e trate adequadamente.
