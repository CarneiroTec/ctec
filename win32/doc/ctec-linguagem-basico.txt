CTEC - Linguagem de Programação Português Brasileiro

Este documento apresenta um tutorial básico das palavras-chave que passaram nos testes. Para cada palavra-chave há um exemplo mínimo e a saída esperada no terminal.

01 - Vazio
Palavra-chave: Vazio
Exemplo de código:
#Inclua<stdio.h>

Vazio f_vazio(){
    Retorne;
}

Inteiro Início(){
    f_vazio();
    printf("01_Vazio OK\n");
    Retorne 0;
}
Saída no terminal:
01_Vazio OK

    Troque (x) { Caso 0: printf("15_Caso 0\n"); Interrompa; Padrão: printf("15_Padrão\n"); }
    Retorne 0;
}
Saída no terminal:

16 - Continue / Interrompa
Palavra-chave: Continue, Interrompa
Exemplo de código:
#Inclua<stdio.h>

Inteiro Início(){
    Inteiro i = 0;
    Enquanto (i < 4) {
        i = i + 1;
        Se (i == 2) Continue;
        Se (i == 3) Interrompa;
        printf("16_i=%d\n", i);
    }
    Retorne 0;
}
Saída no terminal:
16_i=1
16_i=4
